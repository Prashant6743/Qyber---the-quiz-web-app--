# **App Name**: Qyber AI quiz  

## Core Features:

- Quiz Input Form: User interface to input quiz topic, number of questions, and difficulty level (easy, medium, hard).
- AI Quiz Generation: Use a tool to generate quiz questions and answers based on the provided topic, number of questions and difficulty level.
- Quiz Display: Display the generated quiz questions and answers in a clean and readable format.

## Style Guidelines:

- Primary color: Neutral white or light gray for a clean background.
- Secondary color: Use a calming blue (#3498db) for headings and important elements.
- Accent: A vibrant green (#2ecc71) to highlight correct answers and interactive elements.
- Clean, modern layout with clear sections for input form and quiz display.
- Use simple, intuitive icons to represent difficulty levels (e.g., stars or bars).
- Subtle animations to indicate correct/incorrect answers or when generating the quiz.
